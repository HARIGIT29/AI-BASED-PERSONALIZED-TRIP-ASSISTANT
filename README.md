# AI-BASED-PERSONALIZED-TRIP-ASSISTANT
AI-powered full-stack travel planner for India. Features 7 ML algorithms (TF-IDF, K-Means, Dijkstra's, A*), 33+ REST APIs, real-time weather, hotel/restaurant recommendations, AI itinerary generation, budget allocation, PDF export. Built with React 18, Node.js, Express, MongoDB. Integrated with RapidAPI &amp; Google Places. Production-ready.
